Word

The actual word, the moral of the story, the meat of the matter. This is stored as a string. To query it, use the heading WORD.


Length

The length of the word. Stored as an integer. Query using the heading LENGTH.


Freq_KF

The frequency of the word as reported by the Kucera & Francis (1967) Study. Stored as a decimal. Query using the heading Freq_KF. Generally not accepted as a phenomenal measure of true word frequency.


Freq_HAL

The frequency of a word as reported by the HAL Study. Stored as a decimal. Query using the heading Freq_HAL.


Log_Freq_HAL

Log transform of the frequency of a word as reported by the HAL Study. Stored as a decimal. Query using the heading Log_Freq_HAL.


SUBTLWF

Subtitle frequency: word form frequency. Stored as a decimal. Query using the heading SUBTLWF. You'll want to use this or its log transform if word frequency is important to you.


LgSUBTLWF

Log transform of the SUBTL word frequency. Stored as a decimal. Query using the heading LgSUBTLWF.


SUBTLCD

Subtitle frequency: contextual diversity; see Adelman, Brown, & Quesada (2006) for the qualities of this measure. Stored as a decimal. Query using the heading SUBTLCD.


LgSUBTLCD

Log transform of the SUBTL contextual diversity. Stored as a decimal. Query using the heading LgSUBTLCD.


Ortho_N

The number of orthographic neighbors that a word has. Stored as a decimal. Query using the heading Ortho_N.

SQL TOP Clause

The TOP clause allows you to specify the number of records that you would like returned by your query.

SQL SELECT TOP Syntax

SELECT TOP number|percent column_name
FROM table_name
WHERE column_name operator value

Example

Let's say we want the average LgSUBTLWF for words, grouped by length. (That is, the average for len=2, len=3, ... , len=n) But now we only want the five highest averages:

SELECT TOP 5 LENGTH, AVG(LgSUBTLWF) FROM ELP
GROUP BY LENGTH ORDER BY AVG(LgSUBTLWF) DESC

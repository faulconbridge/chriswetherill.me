SQL BETWEEN Operator

The BETWEEN operator is a quick and easy way to select only rows that have a column falling within a certain range of values. The BETWEEN operator is inclusive (at least in this database). It can be used with strings, dates, and numbers.

SQL BETWEEN Syntax

SELECT column_name
FROM table_name
WHERE column_name BETWEEN value1 AND value2

Example

Say we wanted all words with a length between 1 and 4:

SELECT WORD, LENGTH, LgSUBTLWF FROM ELP
WHERE LENGTH BETWEEN 1 AND 4

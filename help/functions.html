SQL Functions

Math is hard. SQL knows this. SQL sympathizes. SQL has functions to make your life easier.

SQL Aggregate Functions

AVG() Returns the average of a column

COUNT() Returns the number of rows

FIRST() Returns the first value in a column

LAST() Returns the last value in a column

MAX() Returns the largest value in a column

MIN() Returns the smallest value in a column

SUM() Returns the sum of a column

SQL Scalar Functions

UCASE() Converts a field to uppercase

LOWCASE() Converts a field to lowercase

MID() Extracts characters from a text field

LEN() Returns the length from a field

ROUND() Rounds a numeric field to the number of decimals specified

NOW() Returns the current system time

FORMAT() Formats how a field is displayed

SQL LIKE Operator

The LIKE operator is used in a WHERE clause to specify a pattern along which to constrain the results that are returned. The percent sign (%) is used here as a wildcard for multiple characters. For a single character wildcard, use the underscore (_).

SQL LIKE Syntax

SELECT column_name(s)
FROM table_name
WHERE column_name liKE 'value'

Example

Let's build on our previous example. Say your last query was too restrictive and instead of words that are only nouns, maybe you would like to have all words that can have noun forms (among others). Your query qould look something like,

SELECT WORD, POS FROM ELP WHERE POS LIKE '%NN%'

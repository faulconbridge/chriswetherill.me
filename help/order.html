SQL ORDER BY Statement

The ORDER BY statement allows you to display your results in ascending or descending order.

SQL ORDER BY Syntax

SELECT column_name(s)
FROM table_name
WHERE column_name operator value
ORDER BY column_name ASC|DESC
GROUP BY column_name

Example

To select words where the length is 6 and the log frequency per million words is between 2 and 3 and display them in reverse alphabetical order,

SELECT WORD, LENGTH, LgSUBTLWF FROM ELP WHERE LENGTH=6
AND LgSUBTLWF BETWEEN 2 AND 3 ORDER BY WORD DESC

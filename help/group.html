SQL GROUP BY Statement

The GROUP BY statement allows you to group your results by the distinct values in a given column. An aggregate function must be used in the SELECT statement if you use a GROUP BY statement. (E.g., AVG(), COUNT(), MAX(), &c. See the SQL Functions entry for a more comprehensive list.)

SQL GROUP BY Syntax

SELECT column_name, aggregate_function(columnl_name)
FROM table_name
WHERE column_name operator value
GROUP BY column_name

Example

Let's say we want the average LgSUBTLWF for words, grouped by length. (That is, the average for len=2, len=3, ... , len=n):

SELECT LENGTH, AVG(LgSUBTLWF) FROM ELP GROUP BY LENGTH

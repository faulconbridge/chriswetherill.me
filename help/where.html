SQL WHERE CLAUSE

The WHERE clause is used to restrict the results returned to only those that meet certain specified criteria. You can use =, >, and < in a WHERE clause.

SQL WHERE Syntax

SELECT column_name(s)
FROM table_name
WHERE column_name operator 'value'

Explanation

WHERE specifies the constraints that you want to place on the values returned. If the column you're restricting along contains strings, it will look like,

SELECT WORD, POS FROM ELP WHERE POS='NN'

However, if you're restricting along an integer, your statement will look like,

SELECT LENGTH FROM ELP WHERE LENGTH=5
